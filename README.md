Step-by-Step Execution
To run the script correctly without file path errors, follow the **Address Bar Trick**:

1. **Download** the `mongodb_assignment.js` file to a folder on your computer.
2. **Open the folder** in Windows File Explorer.
3. **The Address Bar Trick**: 
   - Click on the **Address Bar** at the top of the folder window (where the folder path is shown).
   - Delete the text, type `cmd`, and press **Enter**.
   - This opens the Command Prompt directly in that folder.
4. **Run the Script**:
   Type the following command in the terminal and press Enter:
   ```bash
   mongosh < mongodb_assignment.js
